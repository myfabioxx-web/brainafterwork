# สมองหลังเลิกงาน Website

เว็บ React + Vite สำหรับเพจ "สมองหลังเลิกงาน"

## วิธีรันในเครื่อง

```bash
npm install
npm run dev
```

## วิธี build

```bash
npm run build
```

## วิธี deploy บน Vercel

1. แตกไฟล์ ZIP
2. อัปโหลดโฟลเดอร์นี้ขึ้น GitHub
3. เข้า Vercel แล้วเลือก Import Project
4. Build Command: `npm run build`
5. Output Directory: `dist`
